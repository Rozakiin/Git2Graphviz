\hypertarget{index_autotoc_md0}{}\doxysection{Git2\+Graphviz}\label{index_autotoc_md0}
\mbox{\hyperlink{classGit2Graphviz}{Git2\+Graphviz}} creates a graphviz file representing the history of a Git repository.

Look how easy it is to use\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{\$ ./git2graphviz 'https://github.com/fador/mineserver' mineserver.gv}
\end{DoxyCode}
\hypertarget{index_autotoc_md1}{}\doxysubsection{Features}\label{index_autotoc_md1}

\begin{DoxyItemize}
\item Create graphviz file of repository history
\item Option to specify start and end commits
\end{DoxyItemize}\hypertarget{index_autotoc_md2}{}\doxysubsection{Installation}\label{index_autotoc_md2}
\hypertarget{index_autotoc_md3}{}\doxysubsection{Dependencies}\label{index_autotoc_md3}

\begin{DoxyItemize}
\item \href{https://git-scm.com/}{\texttt{ git}} (tested with version 2.\+17.\+1)
